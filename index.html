Introdução

Um Design System feito para o Ministério Público do Rio Grande do Sul.

Esta é a primeira versão do nosso Design System: uma base técnica e visual compartilhada para construir interfaces digitais consistentes, acessíveis e alinhadas à identidade institucional. Começamos pelos tokens fundamentais e pelos componentes essenciais, com evolução contínua a partir do uso real nos produtos do MP RS.

  • Tokens importados do Figma e compilados via Style Dictionary
  • Componentes React tipados, com testes e documentação
  • Acessibilidade como critério de aceite
